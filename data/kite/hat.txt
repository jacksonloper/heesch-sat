# Polykite Hat (Einstein Aperiodic Monotile)
#
# The "hat" is the famous aperiodic monotile discovered by Smith, Myers, Kaplan,
# and Goodman-Strauss in 2023. It consists of 8 congruent kites (60째-90째-120째-90째)
# arranged in a specific pattern that tiles the plane only aperiodically.
#
# Key properties of the hat:
# - 8 kites total
# - 14 boundary edges (14-sided polygon)
# - Simply connected (no holes)
# - Cannot tile the plane periodically (aperiodic)
# - Causes memory exhaustion in SAT-based tiling solvers due to exponential
#   growth of the search space (no periodic solution exists)
#
# This file contains the hat's coordinates in the heesch-sat kite grid
# coordinate system. The coordinates were identified by:
# 1. Finding all simply connected 8-kite polykites with exactly 14 boundary edges
# 2. Testing each with the SAT solver
# 3. Identifying the unique shape that exhausts solver memory (characteristic
#    of aperiodic tilers, as the solver searches for a periodic solution
#    that doesn't exist)
#
# Note: The SAT solver in heesch-sat is designed to detect isohedral/periodic 
# tilings. For the hat, the solver will run out of memory before finding a
# solution, since no periodic tiling exists. This behavior confirms the hat's
# aperiodic nature.
#
# Format: K x1 y1 x2 y2 x3 y3 x4 y4 x5 y5 x6 y6 x7 y7 x8 y8
#
# The kites and their orientations:
#   (-3,  2) - orientation 0 (E)
#   (-2,  3) - orientation 4 (SW)
#   (-1,  1) - orientation 2 (NW)
#   (-1,  3) - orientation 5 (SE)
#   ( 1,  0) - orientation 0 (E)
#   ( 1,  2) - orientation 3 (W)
#   ( 2,  1) - orientation 4 (SW)
